nugget

Knowing of no wget-style tools that are purely standard WinAPI-based, I thought it would be fun to make my own, and throw in automatic VirusTotal lookups for downloaded files, for good measure. And of course, as I hate bloatware with a passion, I wanted to make something really small yet effective. Some time later, nugget was born. :)

Should be compatible with all modern Windows versions. (and probably even XP)

Features

  • support for both HTTP and HTTPS
  • automatically quarantines (renames) files deemed suspicious by VT
  • special privilege VT API key
